```bash

conda activate hotel-clustering
python run_all.py
```

Or in Google Colab: open `hotel_clustering_final.ipynb` and select **Runtime → Restart and run all**.

## Methods covered

- **Task 1:** Preprocessing pipeline (R0/R1) + K-Means baseline + iK-Means
- **Task 2:** Ward hierarchical + GMM (two distinct families)
- **Task 3:** Internal indices + stability (11 seeds) + sensitivity (scaler + feature ablation)
- **Task 4:** Report + fully reproducible repository
- **Extension A:** Cluster-aware anomaly detection (Jaccard stability R0/R1 = 0.905)
- **Extension B:** Spectral Clustering with eigengap k-selection
- **Extension C:** UMAP-20D preprocessing + trustworthiness benchmarking
