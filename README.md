## Start

Create a "hotel_booking_course_release_v1_pack" folder in root directory and put the dataset csv file inside the folder named (hotel_bookings_course_release_v1.csv)

## Dataset

Available at: https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand/data

## Know more about dataset

Reference: António, N., de Almeida, A., & Nunes, L. (2019). Hotel booking demand datasets. Data in Brief, 22, 41–49. https://doi.org/10.1016/j.dib.2018.11.126
